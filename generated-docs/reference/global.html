LandingPageApp

Public browser API for the landing page enhancements. The interface is exposed as window.LandingPageApp so contributors can re-run specific behaviors after markup changes or use the API in examples.

Type:
  • object
Properties:
Name Type Description
init function

Initializes all progressive enhancement behaviors.
bindSmoothScrolling function

Enables smooth scrolling for same-page anchor links.
bindKeyboardNavigation function

Enables the Shift + Alt + M shortcut for the main content area.
initLazyLoading function

Activates data-src image loading with an Intersection Observer fallback.
updateActiveNavLink function

Synchronizes the active navigation link with the current scroll position.
initMobileMenu function

Reserves the responsive navigation hook used by future mobile-menu work.
bindFormFocusStates function

Adds and removes the focused helper class on form controls.
